# Register

## 1. POST method

회원가입을 통해 유저 정보를 등록합니다.

### 1. Request URL

```
/register
```

### 2. Parameters

key of request body : description

* id : 유저의 email id
* password : 비밀번호
* nickname : 유저의 닉네임
* locationCategory : 유저의 위치
* foodCategory : 유저의 음식 카테고리 취향
* icon : 프로필 아이콘
