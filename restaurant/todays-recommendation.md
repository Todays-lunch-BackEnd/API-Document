# Today's Recommendation

## 1. GET method

오늘의 추천 맛집들의 list를 반환합니다.

### 1) Request URL

```
/restaurants-recmd
```

### 2) Response Body

key : description

* id : 음식점 id
* restaurantImage : 음식점 사진
* restaurantName : 음식점 이름
* rating : 음식점 평점
* introduction : 음식점 한줄평
* restaurantRecmd : 음식점 추천 수
